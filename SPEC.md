# 课堂加分器 - 产品规范文档

## 1. 项目概述

### 项目名称
课堂加分器 (ClassScore)

### 项目类型
Electron 桌面应用程序 + PWA网页版

### 核心功能
悬浮球式课堂实时加减分工具，让老师在上课时无需切换页面，5秒内完成给学生或小组的加分/扣分操作。

### 目标用户
- 中小学教师
- 培训机构讲师
- 课堂主持人

---

## 2. UI/UX 规范

### 2.1 窗口模型

#### 主窗口
- **尺寸**: 420px × 680px (可调整，最小 360px × 500px)
- **样式**: 圆角窗口 (border-radius: 16px)
- **位置**: 屏幕右侧边缘悬浮
- **行为**: 最小化时化为悬浮球

#### 悬浮球
- **尺寸**: 56px × 56px (圆形)
- **位置**: 屏幕右侧边缘居中
- **行为**: 可拖拽、点击展开主窗口
- **效果**: 悬停放大 (scale: 1.1)，点击涟漪效果

## 3. 功能规范

### 3.1 悬浮球模块

#### 功能
- 始终悬浮在屏幕边缘
- 可拖拽到任意位置
- 点击展开主窗口
- 右键菜单 (设置、隐藏、退出)

#### 状态
- 默认: 显示图标
- 悬停: 放大 + 发光效果
- 拖拽中: 半透明
- 消息: 显示红点

### 3.2 加减分模块

#### 操作流程
1. 点击悬浮球 → 弹出操作盘
2. 选择学生/小组 (搜索过滤)
3. 选择分值 (+1/+3/+5/+10/-1/-3/-5/-10 或自定义)
4. 确认 → 显示飘分动画 → 自动消失

#### 分值自定义
- 名称 (如 "回答问题")
- 分值 (-100 ~ +100)
- 图标 (emoji 或内置图标)
- 颜色
- 快捷键

#### 快捷操作
- 快速选择最近操作的学生
- 快捷键加分 (数字键 + 方向键)

### 3.3 学生/小组管理

#### 学生管理
- 添加/编辑/删除学生
- 头像 (emoji 或图片)
- 分组
- 初始分数

#### 小组管理
- 创建/编辑/删除小组
- 成员拖拽分配
- 小组头像/颜色

#### 导入/导出
- CSV 导入学生名单
- 导出规则包 (JSON)

### 3.4 排行榜模块

#### 展示
- 实时排名
- 今日/本周/本月切换
- 趋势箭头 (↑↓→)
- 前三名金/银/铜高亮

#### 动画
- 排名变化平滑过渡
- 新增分数闪烁

### 3.5 数据存储

#### 本地存储
- SQLite 数据库
- 位置: 用户数据目录

#### 数据表
- students (学生)
- groups (小组)
- scores (分数记录)
- rules (分值规则)
- settings (设置)

### 3.6 API 模块

#### RESTful 端点
```
POST /api/score/add      # 加分
POST /api/score/subtract # 扣分
GET  /api/students       # 获取学生列表
GET  /api/groups         # 获取小组列表
GET  /api/ranking        # 获取排行榜
GET  /api/scores         # 获取分数记录
POST /api/students       # 添加学生
PUT  /api/students/:id   # 更新学生
DELETE /api/students/:id # 删除学生
```

#### 认证
- API Key 头部: X-API-Key

#### Webhook
- 配置回调 URL
- 事件: score.added, score.subtracted

## 4. 性能优化

### 4.1 算法优化
- 排行榜使用虚拟列表 (1000+学生)
- 分值变化使用增量更新
- 搜索使用防抖 (300ms)
- 数据库使用索引

### 4.2 渲染优化
- 使用 CSS transform 而非 top/left
- 动画使用 GPU 加速
- 图片懒加载
- 减少重排重绘

### 4.3 内存优化
- 及时清理事件监听
- 使用对象池
- 大列表虚拟化
