# 班级积分系统 - 数据初始化说明

## 问题说明

主窗口的人员名单显示依赖 API 服务器提供的数据。如果 API 服务器中没有数据，主窗口将无法显示小组和学生信息。

## 解决方案

### 方法一：使用设置页面初始化（推荐）

1. 打开主窗口
2. 点击"设置"标签页
3. 滚动到底部，点击"初始化班级数据"按钮
4. 确认后，系统将自动添加：
   - 6 个小组（一组到六组）
   - 33 名学生（每个小组 5-6 人）

### 方法二：在功能页面快速初始化

在以下页面的空状态提示中，都可以点击"初始化班级数据"按钮：

- **加分页面**：当没有小组数据时
- **排行页面**：当没有排行数据时
- **学生页面**：当没有学生数据时

## 快速操作面板

快速操作面板（panel.html）已经内置了预设数据，即使 API 服务器未初始化也能显示人员名单和进行加分扣分操作。

## 注意事项

1. **首次使用**：必须先初始化数据才能在主窗口看到人员名单
2. **重复初始化**：如果数据已存在，系统会跳过已存在的项目，不会重复添加
3. **自定义数据**：初始化后可以在"学生"页面添加、编辑或删除学生和小组
4. **API 服务器**：确保 API 服务器正常运行（端口 3001）
