# 班级积分系统

一个基于 Electron 的桌面应用程序，用于管理班级积分，支持小组和个人分数记录、周清算、随机抽人等功能。

## 🌟 核心特性

### 悬浮球系统 ⚡
- **始终置顶**：悬浮球和快速操作面板永远在最顶层，即使播放PPT时也能使用
- **快速操作**：点击悬浮球立即弹出快速操作面板，5秒内完成加分/扣分
- **可拖拽**：悬浮球可以拖动到屏幕任意位置
- **快捷键**：`Ctrl + Shift + S` 快速显示/隐藏

### 主功能窗口 📊
- 完整的管理界面：加分、排行、周清算、学生管理、设置
- 所有数据与快速操作面板同步
- 可随时打开查看详细信息和历史记录

## 功能特点

### ✨ 核心功能
- **📊 周清算系统**：每周自动清算分数，记录单周得分和累计得分
- **👥 小组管理**：6个小组，每组5-6名学生，带有彩色标识
- **🎯 个人积分**：个人分数记录，支持周统计和累计统计
- **🎲 随机抽人**：一键随机抽取学生进行提问或加分
- **⚡ 快速操作**：悬浮球操作面板，快速加减分
- **🏆 排行榜**：实时排名显示，小组和个人分开排行

### 🎨 界面设计
- 每个小组有独特的彩色标识（紫色、蓝色、绿色、橙色、黄色、红色）
- 浅色背景卡片，视觉效果更柔和
- 清晰的排名显示（🥇🥈🥉）
- 流畅的动画效果

## 使用方法

### 启动应用
1. 双击启动应用
2. 应用启动后会显示：
   - **主窗口**：完整的班级积分管理界面
   - **悬浮球**：屏幕右侧的紫色圆球（可拖动）
   - **系统托盘**：右下角托盘图标

### 悬浮球快速操作 ⚡

#### 打开快速面板
- 方法1：点击悬浮球
- 方法2：按 `Ctrl + Shift + S`

#### 快速加分流程
1. **选择小组**：点击对应颜色的小组卡片
2. **选择组员**（可选）：点击组员列表中的学生
3. **加分/扣分**：
   - 小组操作：`+3×人数` 或 `-2×人数`
   - 个人操作：`+3` 或 `-2`

#### 随机抽人 🎲
1. 点击"🎲 随机抽人"按钮
2. 系统自动选中一名学生
3. 直接进行加分操作

#### 关闭面板
- 点击面板右上角 ✕
- 按 `Esc` 键
- 点击面板外部

### 主窗口使用

#### 首次使用
1. 打开主窗口
2. 点击"周清算"标签页
3. 点击"初始化数据"按钮
4. 系统会自动创建6个小组和32名学生

#### 周清算 📊
1. 每周结束时，进入"周清算"标签页
2. 点击"开始清算"按钮
3. 系统会：
   - 记录本周所有小组和个人的得分
   - 计算排名
   - 重置周分数为0
   - 保留累计分数
4. 可以查看历史周清算记录

#### 排行榜 🏆
- 实时显示小组和个人排名
- 前3名显示金银铜牌
- 显示本周分数和累计分数

## 悬浮球特性说明

### 始终置顶 ⬆️
- 悬浮球和快速操作面板使用 `alwaysOnTop: true` 设置
- 在所有应用（包括PPT、视频、游戏等）上方显示
- 不受其他窗口影响

### 交互设计
- **点击**：打开快速操作面板
- **拖拽**：移动悬浮球位置
- **悬停**：放大动画效果
- **点击面板外**：自动关闭面板

### 适用场景
- 📺 播放PPT时快速加分
- 🎬 播放视频时记录表现
- 🎮 展示课堂互动
- 📱 任何需要快速操作的场景

## 打包 Windows 安装包

### 前置要求
本项目已配置为无需依赖的独立安装包，打包后可直接在 Windows 电脑上运行。

### 打包步骤

#### 方法一：仅打包 Windows 版本
```bash
npm run dist:win
```

#### 方法二：打包所有平台
```bash
npm run dist
```

### 打包产物
打包完成后，会在 `release` 目录下生成以下文件：

1. **安装程序 (.exe)**
   - 文件名：`班级积分系统 Setup 1.0.0.exe`
   - 特点：
     - 标准 NSIS 安装程序
     - 用户可选择安装目录
     - 自动创建桌面快捷方式
     - 自动创建开始菜单项
     - 卸载程序

2. **便携版 (.exe)**
   - 文件名：`班级积分系统-便携版-1.0.0.exe`
   - 特点：
     - 无需安装，双击即可运行
     - 数据保存在程序目录
     - 适合放在U盘使用

### 在无环境依赖的 Windows 电脑上使用

1. **安装版使用**：
   - 双击 `班级积分系统 Setup 1.0.0.exe`
   - 按照安装向导完成安装
   - 桌面会自动创建快捷方式
   - 双击快捷方式即可启动应用

2. **便携版使用**：
   - 将 `班级积分系统-便携版-1.0.0.exe` 复制到任意位置
   - 直接双击运行
   - 无需安装，无需管理员权限

### 注意事项
- 应用已内置所有依赖，无需安装 Node.js 或其他环境
- 数据存储在用户数据目录，卸载后数据不会丢失
- 便携版数据存储在程序目录下，可以随程序一起移动
- **悬浮球始终置顶**，即使在播放PPT时也能使用

## 数据存储

应用使用本地存储，数据位置：
- **安装版**: `%APPDATA%/班级积分系统/`
- **便携版**: 程序目录下 `data/` 文件夹

数据包括：
- 学生信息
- 小组信息
- 分数记录
- 周清算记录
- 设置项

## 快捷键

- `Ctrl + Shift + S`: 显示/隐藏主窗口
- `Esc`: 关闭快速操作面板或返回上一级

## 系统托盘

应用最小化后会在系统托盘显示图标，可以：
- 点击图标显示/隐藏主窗口
- 右键菜单：显示窗口、隐藏窗口、退出

## 常见问题

### Q: 悬浮球不见了怎么办？
A: 检查系统托盘，双击托盘图标或右键选择"显示窗口"

### Q: 悬浮球在PPT播放时无法点击？
A: 确保使用的是应用内置的悬浮球（紫色圆球），而不是浏览器页面

### Q: 如何让悬浮球不遮挡内容？
A: 可以拖拽悬浮球到屏幕边缘或合适的位置

### Q: 两个窗口的数据会同步吗？
A: 是的，悬浮球面板和主窗口的数据是实时同步的
